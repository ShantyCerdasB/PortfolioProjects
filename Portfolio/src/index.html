Work Gallery

Don Emilio Coffee

This project was developed using React, TypeScript, NestJS, and SASS. It is designed for a family in Pérez Zeledón, Costa Rica, with the aim of providing an interactive platform where users can learn more about the products and the story behind the family that produces Don Emilio coffee. The application features product displays, information about the cultivation process, and the ability to directly contact the producers.

Pokémon TCG Card Game

This is a Pokémon TCG (Trading Card Game) developed using TypeScript, CSS, and HTML. The game allows players to engage in a digital card battle experience, where they can collect and battle with Pokémon cards. The game simulates various aspects of the Pokémon TCG, including energy management, attack strategies, and special abilities.
